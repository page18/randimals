About

This website is a live demo of the Randimals package, available on NPM.

To use this demo, set the options below and then hit the submit button to generate a unique and adorable animal-based ID.

Options

Example: 2 = Colorful Mousey Elephant

Example: 2 = Aggressive Tiger Whale

Example: snake = ghastly_otter Example: pascal = AggressiveTigerWhale

Example: UPPERCASE and , = SPECTACULAR,RHINOCEROS Example: Capitalized and + = Pompous+Monkfish

NOTE: Using the case or separator option(s) will override the format option, using the given case and separator instead.
